# プラン曖昧点の明確化

## 手順

1. プランファイルを読んで曖昧点をリストアップ
2. `AskUserQuestion` で質問（2〜4問、各選択肢にpros/cons付き）
3. 決定内容をプランファイルに書き込む
4. 未解決の曖昧点があれば2に戻る

## 質問設計の注意点

- CLAUDE.mdの既存パターン・技術スタックに沿った選択肢を作る
- オープンエンド質問禁止（選択肢を必ず用意）
- `multiSelect: true` は選択肢が排他的でない場合のみ使用
